## 🏷️ Branding & Logo

### Logo Plesir Pekalongan
- [ ] Logo full color (SVG + PNG)
- [ ] Logo monochrome (untuk dark background)
- [ ] Logo favicon (192x192, 512x512)
- [ ] Logo maskable untuk PWA
- [ ] Open Graph image (1200x630)

**Konsep logo:**
- Gabungan motif batik pesisir + ombak laut
- Warna: deep ocean (#0A4D68) atau teal (#088395)
- Font: Playfair Display untuk "Plesir Pekalongan"

**Tools gratis untuk buat logo:**

## 💡 MVP Strategy: Placeholder Saja Dulu

Untuk Phase 1 (MVP), pakai placeholder:

1. **Unsplash** untuk foto: `https://source.unsplash.com/800x600/?pekalongan,batik`
2. **Lucide React** untuk semua icons
3. **DiceBear** untuk avatar dummy
4. **Logo:** generate dengan Canva atau pakai text-only

Setelah Phase 1 stabil, baru ganti dengan asset asli untuk Phase 2 (production).
